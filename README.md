# Frontend Mentor - FAQ accordion card solution

This is a solution to the [https://www.frontendmentor.io/challenges/faq-accordion-card-XlyjD0Oam]. Frontend Mentor challenges help you improve your coding skills by building realistic projects. 

### The challenge

Users should be able to:

- View the optimal layout for the component depending on their device's screen size
- See hover states for all interactive elements on the page
- Hide/Show the answer to a question when the question is clicked

### What I learned

CSS:
Apparently it is not possible to set a transition between display none and display block. Using visibility and height can be an alternative.

JS:
I used the nextElementSibling property for the first time.
Note: there is also a previousElementSibling property.

Reminder: with arrow functions there is no binding of 'this'.
That's why I used a declared function for the event.
